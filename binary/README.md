# Binaries

This directory contains prebuilt binary of terminal-idle.

They are intended for convenience and quick testing of the feature.

⚠️ Use at your own discretion and ensure you trust the provided builds before installing.

---
Copy `terminal-idle` to ~/.local/bin directory  
Copy `terminal-idle.service` to ~/.config/systemd/user  

Register `terminal-idle.service` and enjoy!
